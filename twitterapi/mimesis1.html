About

What is Mimesis 1.0
Mimesis 1.0 is a website that conducts digital profiling as a third party. Twitter is the data source platform. In the 'Generate your Mimesis' section, input the Twitter ID and Count to generate more Mimesis 1.0 profile. There is an option to upload the new profile to Twitter: Mimesis.


How it works?
Tweets are pulled from usersâ€™ home-timeline by Twitter API and piped into a sentiment analysis model which will spit out 1,0,-1 meaning positive, neutral, and negative which will be matched to green, blue, and red. Circles are drawn from inside to the outside.

As a tree ring storing climate and atmospheric conditions data, Mimesis 1.0 stores personal sentiment data. It is a visual representation of the internet user. It is an avatar, as well as simulacra.


How to 'read' Mimesis 1.0?
This report is one of the outcomes of the Mimesis project. It is a reference for you to parse, understand, and digest Mimesis 1.0. In this report, the Twitter account is Gigi Hadid, and the Count is 200.
